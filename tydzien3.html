Jak działa internet

HTTP(S) - Hypertext Transfer Protocol (Secure) - protokół, którego używa przeglądaka do transportowania HTML-a. Pobiera i wyświetla pliki znajdujące się w stronach internetowych.

URL - Uniform Resource Locator - adres strony. Ten adres składa się z kilku części:

https:// cotenfrontend :443 / index.html ?parametry=wartość

  • https:// - protokół
  • cotenfrontend.pl - host (adres serwera)
  • :443 - port (ta część zazwyczaj jest dla użytkownika niewidoczna). Wskazuje przeglądarce, na którym porcie na serwerze znajdują się pliki strony
  • / - bardzo ważny znak / - root - przy podawaniu ścieżki do zasobów (src). Jeśli podajemy źródło ze slashem na początku (src="/img.jpg"), przeglądarka szuka w katalogu głównym. Jeśli podajemy bez (src="img.jpg") - szuka w katalogu aktualnym, czyli np w katalogu podstrony. Dlatego lepiej podawać źródło (na serwerze) ze slashem na początku.
  • index.html - ścieżka dostępu do zasobu
  • ?paramert=wartość - parametry

DNS - Domain Name System - system tłumaczący nazwę mnemoniczną (czyli "dla ludzi") na odpowiadający jej adres IP.

parsowanie - sprawdzanie składni

Kody odpowiedzi protokołu HTTP:

  • 10x - czekaj
  • 200 - wszystko ok.
  • 30x - Przekierowania: szukasz w złym miejscu - przekieruję cię w dobre, bo wiem, gdzie.
  • 404 - nie mam tego, czego potrzebujesz :(
  • 50x - błędy serwera

Kody możemy sprawdzić w Devtools (F12) w zakładce Network.

JavaScript

Kod javascript moge umiescic bezposrednio w pliku HTML lub w pliku zewnętrznym. kod JS dodaje w znacznikach <script></script> i dodaje go na samym dole pliku HTML za <footer> a tuz przed </body>

<script>
alert('siema');
</script>

wyskoczy okienko z napisaem 'siema' PRZED zaladowaniem sie strony.

Mozemy zapisac kod JS w pliku z rozszezeniem .js, np main.js. Wtedy w znacznikach na koncu strony wpisuje :

<script src="main.js"></script>
czyli podaje link do pliku main.js jako argument znacznika script i nie uzyje href= jak w stylach, tylko src= tak jak w obrazkach.

POLECENIE:

kolejnosc wykonywania zadan w kodzie js jest taka jak kolejnosc kodu, poprostu czytany jest kod linijka po linijce

POLECENIA NA MACA I NA LINUXA

  • cd - wejscie do katalogu, jesli dobijesz klawiszem tab to dopelni nazwe katalogu
  • ls - lista katalogów
  • ls -l lista katalogow w formie listy
  • cd .. przenosi nas wyzej (wychodze z katalogu)
  • cd ../.. - przenosi mnie dwa katalogi wyzej
  • mkdir - tworzy katalog
  • pwd - pokaze pełną sciezke do katalogu w ktorym sie znajduje
  • code . -(code spacja kropka) - uruchamia nam VSC w tym katalogu w ktorym jestem wpisujac "code ."

  • cd x tab (x pierwsza litera szukanego folderu) wciskając tab'ulator windows podpowiada jakie są katalogi na litere 'x'
  • cd .. przenosi nas wyzej (wychodze z katalogu)
